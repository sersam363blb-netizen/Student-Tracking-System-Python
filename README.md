# University Student Tracking System (CLI)

A console-based student management system developed using **Python** and **SQLite**, designed to perform basic administrative operations on student records.

## Overview
This project allows managing university students through a command-line interface, including creating, updating, deleting, and listing student records stored in a local database.

## Features
- Add new students
- Update student information
- Delete student records
- List students by:
  - Faculty
  - Department
  - Education type
  - Status
- Persistent data storage using SQLite
- Object-oriented design

## System Structure
- Python (CLI-based application)
- SQLite database
- Single-table student schema
- Menu-driven interaction flow

## Technologies Used
- Python
- SQLite (sqlite3)

## Notes
This project was developed for educational and portfolio purposes to demonstrate database operations and object-oriented programming in Python.
